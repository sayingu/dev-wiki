# ES6 Features
## Variable Scoping
```
{
    let a = 7;
    console.log(a);
}
```
### let keyword
```
let a = 8;
let b = 9;
```

### const keyword
```
const c = 10;
```

## Template Literals
```
console.log(`${a} ${b}`);
```
